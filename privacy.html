🐱 고양이 키우기 — 개인정보 처리방침

시행일: 2026-06-12 · 적용 대상: macOS 앱 "고양이 키우기", Windows 앱 "Desktop Cat"

1. 요약

이 앱은 자체 서버가 없으며, 개인정보를 수집·저장·전송하지 않습니다. 고양이 상태, 이름, 사료 재고 등 모든 게임 데이터는 사용자의 기기에만 저장됩니다.

2. 기기 안에서만 처리되는 정보

  • 게임 저장 데이터 — 고양이 스탯·이름·오브젝트 배치 등은 로컬 파일로만 저장되며 외부로 전송되지 않습니다.
  • 알림·Dock 배지 읽기 (macOS, 선택) — "알림 물어오기" 기능을 켜면 손쉬운 사용(Accessibility) 권한으로 화면의 알림 배너·배지 텍스트를 읽습니다. 이 내용은 고양이 연출에만 쓰이고 즉시 버려지며, 저장·전송되지 않습니다.
  • Claude Code 세션 감지 (선택) — 세션 고양이 기능은 ~/.claude/projects 폴더의 파일 수정 시각과 폴더 이름만 로컬에서 읽습니다. 파일 내용은 읽지 않으며 아무것도 전송되지 않습니다.
  • 미디어 재생·배터리·자리비움 상태 — 고양이 반응 연출을 위해 기기에서만 확인합니다.

3. 제3자 서비스

  • 웹사이트 분석 (Google Analytics) — 다운로드 페이지 방문 수와 다운로드 버튼 클릭 수를 파악하기 위해 Google Analytics를 사용할 수 있습니다. 이 정보는 웹사이트 개선과 배포 현황 확인에만 사용됩니다. Google 개인정보처리방침
  • 결제 (Gumroad) — 품종 입양 결제는 브라우저의 Gumroad 페이지에서 진행됩니다. 이메일·결제 정보는 Gumroad가 직접 처리하며 앱은 라이선스 키의 유효성만 Gumroad API로 확인합니다. Gumroad 개인정보 처리방침
  • 광고 (GameMonetize) — "광고 보기"는 브라우저에서 광고 포함 게임 페이지를 엽니다. 광고 쿠키·식별자는 해당 페이지와 광고 사업자의 정책을 따릅니다. GameMonetize 정책
  • 업데이트 확인 (GitHub Pages) — 새 버전 확인을 위해 버전 정보 파일을 내려받습니다. 이때 GitHub의 서버 로그에 IP 주소가 남을 수 있습니다 (GitHub 정책 적용).

4. 어린이

앱 자체는 개인정보를 수집하지 않으며, 결제·광고는 모두 외부 브라우저에서 성인 보호자의 결제 수단/판단이 필요한 형태로 진행됩니다.

🐱 Desktop Cat — Privacy Policy

Effective: 2026-06-12 · Applies to: "고양이 키우기" for macOS and "Desktop Cat" for Windows

1. Summary

This app has no server of its own and does not collect, store, or transmit any personal data. All game data (cat stats, names, item placement) is stored only on your device.

2. Data processed only on your device

  • Game save data — stored in local files only; never uploaded.
  • Notification & Dock badge reading (macOS, optional) — with Accessibility permission, the app reads on-screen banner/badge text solely to animate the cats. The text is discarded immediately and never stored or sent.
  • Claude Code session detection (optional) — reads only file modification times and folder names under ~/.claude/projects, locally. File contents are never read; nothing is transmitted.
  • Media playback / battery / idle state — checked locally to drive cat reactions.

3. Third-party services

  • Website analytics (Google Analytics) — The download page may use Google Analytics to understand page visits and download button clicks. This is used only to improve the website and understand distribution. Google Privacy Policy
  • Payments (Gumroad) — breed adoptions are purchased on Gumroad in your browser. Your email and payment details are handled by Gumroad; the app only verifies license keys via the Gumroad API. Gumroad Privacy Policy
  • Ads (GameMonetize) — "Watch ad" opens an ad-supported game page in your browser; ad cookies and identifiers are governed by that page and its ad providers. GameMonetize policy
  • Update check (GitHub Pages) — the app downloads a version file to check for updates; your IP address may appear in GitHub's server logs (GitHub's policies apply).

4. Children

The app itself collects no personal data; purchases and ads happen in an external browser and require an adult's payment method and judgment.
